# FUTURE_ML_01
**Sales Forecasting Dashboard**

## Project Overview
This project is part of my Machine Learning Internship at Future Interns (Track: Machine Learning). 
I built a dashboard using Python to predict and visualize future sales trends based on historical retail data.

## Files Included
* `task1.py`: The Python script containing the forecasting logic.
* `sales forecast graph.png`: The output visualization of sales trends.
* `train.csv`: The dataset used for training the model.

## Tools Used
* **Language:** Python
* **Libraries:** Pandas, Matplotlib
* **IDE:** Visual Studio Code
